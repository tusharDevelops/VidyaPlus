## 🎨 Design System at a Glance

### Color Palette
```
Primary: Indigo-600 (#4f46e5)
Secondary: Blue-600 (#2563eb)
Accent: Yellow (#fccb04)
Dark BG: Slate-900 (#0f172a)
Text: Gray scale (various levels)
```

### Typography
```
Headings: 900 weight, 1.2 line-height
Body: 500-600 weight, 1.5 line-height
Labels: 700 weight, 1.25 line-height
```

### Spacing
```
Base Unit: 4px
Common: 12px, 16px, 24px, 32px, 48px
```

### Animations
```
Quick: 200ms (interactions)
Standard: 300ms (hover effects)
Smooth: 500ms (transitions)
Easing: cubic-bezier(0.4, 0, 0.2, 1)
```
